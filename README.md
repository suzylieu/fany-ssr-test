
### env setup
1. install nodejs (https://nodejs.org)
2. install npm (version 6.14.12)
3. install yarn (version 1.22.10) 
4. 用 yarn 去跑專案比較快
5. 在專案資料夾下輸入 yarn 或 npm install
6. yarn dev
